ЗАДАНИЕ 

Реализуйте одностороннюю функцию быстрого возведения в степень.
Выполнение программы должно сопровождаться подробной трассировкой (см. таблица из лекции)
С помощью переменной-счетчика определите фактической количество выполненных умножений. Проверьте результат при аргументах с разным весом Хэмминга.
Требование – нельзя использовать встроенные функции, например, pow(a, x, m) в Python.
Подобные функции можно использовать только для проверки.

Требования к реализации:
- можно выбрать любой язык программирования;
- UI – любой (консоль, GUI, чат-бот и пр.);
- тип программного приложения – любой (консоль, desktop, web, мобильное);
- демонстрацию работы программы записать в виде GIF-файла;
- программный код и GIF разместить в репозитории GIT.
