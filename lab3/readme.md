ЗАДАНИЕ. ШИФР ВЕРНАМА И СОВРЕМЕННЫЕ ПОТОЧНЫЕ ШИФРЫ 
Цель задания — программно реализовать поточные шифры.
Задачи 
1. Сгенерируйте файл из случайных символов с помощью встроенного ГПСЧ или самостоятельно реализованного линейного конгруэнтного генератора. В следующем пункте этот файл будет ключом шифра Вернама. 
2. Реализуйте шифр Вернама для зашифрования/расшифрования файла Выбрать файл с открытым текстом и файл с ключом (сгенерирован в предыдущем задании). Результатом должен стать новый файл (XOR первых двух файлов). Аналогично реализуйте расшифрование.
З. Реализуйте применение любого поточного шифра. Можно написать простой шифр самостоятельно (например, RС4) или взять готовую реализацию из какой-либо криптографической библиотеки.

Требования к реализации:
- можно выбрать любой язык программирования;
- UI – любой (консоль, GUI, чат-бот и пр.);
- тип программного приложения – любой (консоль, desktop, web, мобильное);
- демонстрацию работы программы записать в виде GIF-файла;
- программный код и GIF разместить в репозитории GIT.
